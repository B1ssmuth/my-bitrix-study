# Bitrix Study Project (OTUS)

Учебный проект по разработке на платформе Bitrix D7. Основной упор сделан на чистоту кода, использование современных стандартов Bitrix ORM и архитектурный подход.

## 🛠 Технологический стек
- **PHP 8.2** (с использованием PSR-4 автозагрузки)
- **Bitrix D7 Framework**
- **MySQL / MariaDB** (ORM DataManager)
- **Docker** (BitrixDock)

## 📁 Структура проекта
- `/local/App/Models/` — Кастомные модели данных и сущности ORM.
- `/local/App/Models/Lists/` — Модели для работы со свойствами инфоблоков.
- `/homeworks/` — Реализация домашних заданий.

## 🚀 Реализованные модули
- **Управление медперсоналом**: Инфоблоки «Врачи» и «Процедуры» с раздельной таблицей свойств (Версия 2.0).
- **ORM Журнал**: Кастомная таблица в БД, связанная с инфоблоками через Reference-поля.
- **Оптимизация**: Использование runtime-полей и кэширования выборок.
